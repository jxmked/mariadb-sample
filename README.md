# mariadb-sample

Using single database to sync two interfaces (Web, CLI). 

## Requirements
 
**Environments and module/libraries**

  - Atleast `Python` 3.10
  - Atleast `PHP` 8.0
  - Atleast `NodeJS` 16.18
  - Atleast `mariadb` 10.9
